# TransitOS Cloud Geçiş Planı

## Aşama 1 - Web Panel Temeli

- Next.js panel kurulumu
- PostgreSQL veri tabanı
- Yönetici ve taşeron girişleri
- Dashboard özetleri
- Taşeron, araç, proje, güzergah, mesai, gider ve hakediş tabloları

## Aşama 2 - Operasyon Ekranları

- Taşeron yönetimi
- Araç ve şoför yönetimi
- Proje ve güzergah planlama
- Takvim üzerinden tekil mesai ekleme
- Giderlerin aylık hakedişten düşülmesi

## Aşama 3 - Rota ve Raporlama

- Harita üzerinde durak ekleme
- Yol bazlı rota hesaplama
- Tahmini kilometre
- Hakediş PDF çıktısı
- Proje PDF çıktısı
- Rota PDF çıktısı

## Aşama 4 - Mobil Kullanım

- Responsive web arayüzü
- PWA kurulumu
- iOS ve Android ana ekrana ekleme desteği
- Daha sonra istenirse native iOS/Android uygulama

## Aşama 5 - macOS Uygulaması Bağlantısı

- Mevcut SwiftUI uygulamasına API istemcisi eklenir.
- SwiftData sadece offline cache olarak kalır.
- Ana veri kaynağı TransitOS Cloud olur.
